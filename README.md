# aws-secure-ec2-lab
This project demonstrates a secure AWS EC2 deployment using IAM roles, restricted SSH access, and logging via CloudTrail.
Objectives
- Deploy EC2 securely
- Apply least privilege IAM
- Prevent public exposure
- Enable logging and monitoring

Architecture

Services Used
- EC2
- IAM
- S3
- CloudTrail
- CloudWatch

Security Implementations
- IAM role instead of access keys
- Security group restricted to my IP
- S3 public access blocked
- CloudTrail enabled for auditing
